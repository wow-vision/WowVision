# Keybindings

WowVision is fully keyboard-driven, and most of its keybindings will feel natural if you've used a screenreader before. The essentials are covered in [Using the Interface](./using-the-interface.md) and [Buffers](./buffers.md), but here's a quick recap of the most important ones:

**Interface Navigation:**
- **Arrow keys** — move between elements
- **Tab / Shift+Tab** — next / previous element
- **Enter** — left click
- **Backspace** — right click
- **Space** — read tooltip
- **Shift+Arrow keys** — navigate tooltip lines
- **Escape** — close window
- **Ctrl+Tab / Ctrl+Shift+Tab** — switch between open windows

**Buffers:**
- **Alt+Up / Alt+Down** — cycle buffer items
- **Alt+Left / Alt+Right** — switch buffers
- **Alt+Ctrl+Left / Alt+Ctrl+Right** — switch buffer groups

## Customizing Keybindings

You can view and change any keybinding by typing `/wv bind` in chat. This opens the keybinding configuration window, where you can see all available bindings across every module and rebind them to whatever you prefer.

WowVision also supports additional keybindings beyond the defaults — individual modules may offer extra bindings for specific features that aren't bound to a key by default. The bind menu is the best way to discover what's available.

## A Note on Game Keybindings

WowVision's menu navigation keybindings (arrow keys, Tab, Enter, etc.) are only active when a WowVision window is open. When no window is open, your normal WoW keybindings work as usual, so there's no conflict with your regular gameplay keys.

Other keybindings like buffers work at any time, whether a window is open or not.
