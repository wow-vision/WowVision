# Commands

WowVision adds several slash commands. Type these in WoW's chat box.

## WowVision Menu

| Command | Description |
|---|---|
| `/wv` | Open the WowVision menu |
| `/wv bind` | Open the keybinding configuration window |
| `/wv version` | Show the current WowVision version |
| `/wv close` | Force close the WowVision UI |

## Focus Targets

WoW only supports one focus target by default. WowVision extends this with up to five virtual focus targets that can be used throughout the addon.

| Command | Description |
|---|---|
| `/focus1` through `/focus5` | Set a focus target to your current target, or specify a unit (e.g., `/focus1 party1`) |

## Utility

| Command | Description |
|---|---|
| `/pquit` | Leave your current party |
| `/dquit` | Leave your current dungeon or LFG group |
| `/enableaddon <name>` | Enable an addon and reload the UI |
| `/disableaddon <name>` | Disable an addon and reload the UI |
